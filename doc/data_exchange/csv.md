# Описание CSV обмена

Для обмена данными с терминалом сбора данных (ТСД) предлагается следующий формат на основе CSV-файлов.
Он разделен на два основных потока: загрузка справочной информации в ТСД и выгрузка созданных документов.

Первое поле каждой строки файла является ключем 

## Справочная информация для загрузки в ТСД


| Имя ключа | Описание                       |
|-----------|--------------------------------|
| `N`       | Запись вида аналитики          |
| `A`       | Запись аналитики               |
| `O`       | Запись вида операций           |
| `OP`      | Запись настройки вида операций |
| `G`       | Запись товара                  |
| `S`       | Запись ассортимента поставщика |
| `Z`       | Запись заголовка заказа        |
| `D`       | Запись строки заказа           |


### Ключ (`N`)
Вид аналитики (поставщик, зона, комиссия и др.) 

| № Поля | Тип      | Обязательное | Описание                    | Пример значения |
|-------:|:---------|:-------------|:----------------------------|:----------------|
|      1 | `string` | Да           | Код вида аналитики          | "ПС"            |
|      2 | `string` | Да           | Наименование вида аналитики | "Поставцик"     |


```kotlin
N;"ПС";"Поставщик"
N;"СК";"Склад"
N;"ЗН";"Зона"
```

### Ключ (`A`)
Аналитика

| № Поля | Тип      | Обязательное | Описание               | Пример значения |
|-------:|:---------|:-------------|:-----------------------|:----------------|
|      1 | `string` | Да           | Код аналитики          | "ПС0001"        |
|      2 | `string` | Да           | Наименование аналитики | "Поставщик 1"   |

```kotlin
A;"ПС0001";"Поставщик 1"
A;"ПС0002";"Поставщик 2"
A;"СК0001";"Склад 1"
```

### Ключ (`O`)
Вид операции (приход, расход, ценники, ивентаризация и д.р.)

| № Поля | Тип       | Обязательное | Описание                            | Пример значения |
|-------:|-----------|--------------|-------------------------------------|-----------------|
|      1 | `string`  | Да           | Код операции                        | "01"            |
|      2 | `string`  | Нет          | Код операции для выгрузки документа | ""              |
|      3 | `string`  | Да           | Наименование операции               | "Приход"        |
|      4 | `string`  | Нет          | Код вида аналитики 1                | "ПС"            |
|      5 | `string`  | Нет          | Код вида аналитики 2                |                 |
|      6 | `string`  | Нет          | Код вида аналитики 3                |                 |
|      7 | `integer` | Нет          | Флаги                               | 64              |

|  `preferences` | `VopPreferences` | Нет          | Настройки операции                       |                 |

```kotlin
O;"03";;"Приход";"ПС";;;4
```

### Ключ (`OP`)
Настройка операции

| № Поля | Тип      | Обязательное | Описание     | Пример значения |
|-------:|----------|--------------|--------------|-----------------|
|      1 | `string` | Да           | Код операции | "01"            |
|      2 | `string` | Да           | Имя ключа    | "groupItems"    |
|      3 | `string` | Да           | Значение     | true            |

Список допустимых ключей

| `controlUnitLoad`                   | `boolean` | Контроль грузовых единиц                                        |

| `zeroQuantityDefault`               | `boolean` | Нулевое количество по умолчанию                                 |
| `quantityFromCapacity`              | `boolean` | Подставлять количество из емкости упаковки                      |
| `allowZeroQuantity`                 | `boolean` | Разрешить ввод нулевого количества                              |
| `noRequestQuantity`                 | `boolean` | Не запрашивать количество                                       |
| `allowNewItem`                      | `boolean` | Разрешить воод нового товара                                    |
| `requestNPP`                        | `boolean` | Воод порядкового номера                                         |
| `requestPrice`                      | `boolean` | Ввод поля "Цена"                                                |
| `requestDate1`                      | `boolean` | Ввод поля "Дата1"                                               |
| `requiredDate1`                     | `boolean` | Обязательный ввод поля "Дата1"                                  |
| `requestDate1ForWeightItem`         | `boolean` | Запрос Дата1 только для весового товара                         |
| `banExpiredDate1`                   | `boolean` | Запрет ввода даты1 меньше текущей                               |
| `requestDate2`                      | `boolean` | Ввод поля "Дата2"                                               |
| `requiredDate2`                     | `boolean` | Обязательный ввод поля "Дата2"                                  |
| `banExpiredDate2`                   | `boolean` | Запрет ввода даты2 меньше текущей                               |
| `requestField1`                     | `boolean` | Ввод дополнительного поля 1                                     |
| `requestField2`                     | `boolean` | Ввод дополнительного поля 2                                     |
| `requestField3`                     | `boolean` | Ввод дополнительного поля 3                                     |
| `requiredField1`                    | `boolean` | Обязательное поле 1                                             |
| `requiredField2`                    | `boolean` | Обязательное поле 2                                             |
| `requiredField3`                    | `boolean` | Обязательное поле 3                                             |
| `requestExtraQuantity`              | `boolean` | Запрос дополнительного количества                               |
| `banEditQuantity`                   | `boolean` | Запретить редактировать количество позиции документа            |
| `banEditDocumentNumber`             | `boolean` | Запретить редактировать номер документа                         |
| `banFractQuantityForPieceItem`      | `boolean` | Запрет дробного количества для штучного товара                  |
| `voicePromoItem`                    | `boolean` | Озвучивать акционный товар                                      |
| `hidePrice`                         | `boolean` | Не показывать цену товара                                       |
| `hideBalance`                       | `boolean` | Не показывать остаток товара                                    |
| `withBatches`                       | `boolean` | Работа с партиями                                               |
| `allowTrustAccept`                  | `boolean` | Разрешить доверительную приемку                                 |
| `trustAcceptPercent`                | `integer` | Процент доверительной приемки                                   |
| `isOrder`                           | `boolean` | Необходим выбор заказа для документа                            |
| `banItemOutsideOrder`               | `boolean` | Запрет добавления товара вне заказа                             |
| `quantityFromOrder`                 | `boolean` | Подставлять количество из заказа                                |
| `controlOrderQuantity`              | `boolean` | Контроль количества при вводе                                   |
| `noGroupItemOrder`                  | `boolean` | Не группировать по товару в заказе                              |
| `selectOrdersForOperation`          | `boolean` | Выбор заказов для операции                                      |
| `singleChoiceOrder`                 | `boolean` | Одиночный выбор заказа                                          |

```kotlin
P;"01";"groupItems";true
P;"01";"isMarking";true
```

### Ключ (`G`)
Товар

| № Поля | Тип       | Обязательное | Описание                      | Пример значения                        |
|-------:|-----------|--------------|-------------------------------|----------------------------------------|
|      1 | `string`  | Да           | Штрихкод                      | "4810268024917"                        |
|      2 | `string`  | Да           | Наименование                  | "Молоко пастеризованное, 3.1%, 1 литр" |
|      3 | `double`  | Нет          | Цена                          | 3.35                                   |
|      4 | `double`  | Нет          | Остаток                       | 10                                     |
|      5 | `string`  | Нет          | Доп. поле 1                   | "Беларусь"                             |
|      6 | `string`  | Нет          | Доп. поле 2                   | "ОАО Савушкин продукт"                 |
|      7 | `string`  | Нет          | Доп. поле 3                   | ""                                     |
|      8 | `string`  | Нет          | Доп. поле 4                   | ""                                     |
|      9 | `string`  | Нет          | Доп. поле 5                   | ""                                     |
|     10 | `string`  | Нет          | Имя файла изображения         | "4810268024917.jpg"                    |
|     11 | `double`  | Нет          | Процент доверительной приемки | 10.0                                   |
|     12 | `integer` | Нет          | Флаги  (сумма флагов)         | 64                                     |

`Флаги`:
- 1 - Обязательный ввод даты 1
- 2 - Обязательный ввод даты 2
- 4 - Обязательный ввод дополнительного поля 1
- 8 - Обязательный ввод дополнительного поля 2
- 16 - Обязательный ввод дополнительного поля 3
- 32 - Признак товара интернет магазина
- 64 - Признак маркированного товара
- 256 - Признак акционного товара
- 1024 - Подлежит прослеживанию

```kotlin
G;4810268024917;"Молоко пастеризованное, 3.1%, 1 литр";3.35;10;"БеларусьДобрада";"ОАО Савушкин продукт";;;;"4810268024917.jpg";10;64
```

### Ключ (`S`)
Ассортимент посавщика

| № Поля | Тип      | Обязательное | Описание        | Пример значения |
|-------:|----------|--------------|-----------------|-----------------|
|      1 | `string` | Да           | Код постащика   | "ПС0001"        |
|      2 | `string` | Да           | Штрихкод        | "4810268024917" |
|      3 | `double` | Нет          | Цена            | 3.35            |

```kotlin
S;"ПС0001";"4810268024917";3.35
```


### Ключ (`Z`)
Заказ

| № Поля | Тип      | Обязательное | Описание               | Пример значения       |
|-------:|----------|--------------|------------------------|-----------------------|
|      1 | `string` | Да           | Дата заказа (ISO 8601) | "2026-06-15T11:00:00" |
|      2 | `string` | Да           | Номер заказа           | "123"                 |
|      3 | `string` | Нет          | Код поставщика         | "ПС0001"              |
|      4 | `string` | Да           | Штрихкод товара        | "4810268024917"       |
|      5 | `double` | Нет          | Количество             | 3.0                   |
|      6 | `double` | Нет          | Цена                   | 3.35                  |


```kotlin
Z;"2026-06-25T11:00:00";"123";"ПС0001";"4810268024917";3.0;3.35
Z;"2026-06-25T11:00:00";"123";"ПС0001";"4810268020827";1.0;2.53
```

## Документ ТСД

### Ключ (`D`)
Заголовок документа

| № Поля | Тип      | Обязательное | Описание          | Пример значения       |
|-------:|----------|--------------|-------------------|-----------------------|
|      1 | `string` | Да           | Код вида операции | "01"                  |
|      2 | `string` | Да           | Дата документа    | "2026-06-27T15:00:00" |
|      3 | `string` | Да           | Номер документа   | "12345"               |
|      4 | `string` | Нет          | Код аналитики 1   | "ПС0001"              |
|      5 | `string` | Нет          | Код аналитики 2   |                       |
|      6 | `string` | Нет          | Код аналитики 3   |                       |
|      7 | `string` | Нет          | Комментарий       | "тест"                |

### Ключ (`P`)
Строка документа

|  № Поля | Тип       | Обязательное | Описание             | Пример значения       |
|--------:|-----------|--------------|----------------------|-----------------------|
|       1 | `integer` | Нет          | Порядковый номер     | 0                     |
|       2 | `string`  | Да           | Время ввода записи   | "2026-06-27T15:10:00" |
|       3 | `string`  | Да           | Штрихкод товара      | "4810268024917"       |
|       4 | `double`  | Да           | Количество           | 2                     |
|       5 | `double`  | Да           | Количество по заказу | 2                     |
| `price` | `double`  | Да           | Цена                 | 3.35                  |

```kotlin
D;01;"2026-06-27T15:00:00";"12345";"ПС0001";;;"тест"
P;1;"2026-06-27T15:10:00";"4810268024917";2;2;3.35
P;2;"2026-06-27T15:15:00";"4810268020827";3;5;2.53
```
