# Описание CSV обмена

Для обмена данными с терминалом сбора данных (ТСД) предлагается следующий формат на основе CSV-файлов.
Он разделен на два основных потока: загрузка справочной информации в ТСД и выгрузка созданных документов.

Первое поле каждой строки файла является ключем 

## Справочная информация для загрузки в ТСД


| Имя ключа | Описание                       |
|-----------|--------------------------------|
| `N`       | Запись вида аналитики          |
| `A`       | Запись аналитики               |
| `O`       | Запись вида операций           |
| `G`       | Запись товара                  |
| `S`       | Запись ассортимента поставщика |
| `Z`       | Запись заголовка заказа        |
| `D`       | Запись строки заказа           |

### Ключ (`N`)
Вид аналитики (поставщик, зона, комиссия и др.) 

| № Поля | Тип       | Обязательное | Описание                    | Пример значения |
|-------:|:----------|:--------------|:-----------------------------|:-----------------|
|      1 | `string`  | Да           | Код вида аналитики          | "ПС"            |
|      2 | `string`  | Да           | Наименование вида аналитики | "Поставцик"     |


```kotlin
N;"ПС";"Поставщик"
N;"СК";"Склад"
N;"ЗН";"Зона"
```

### Ключ (`A`)
Аналитика

| № Поля | Тип      | Обязательное | Описание               | Пример значения                      |
|-------:|:----------|:--------------|:------------------------|:--------------------------------------|
|      1 | `string` | Да           | Код аналитики          | "ПС0001" |
|      2 | `string` | Да           | Наименование аналитики | "Поставщик 1"                        |

```kotlin
A;"ПС0001";"Поставщик 1"
A;"ПС0002";"Поставщик 2"
A;"СК0001";"Склад 1"
```

### Ключ (`O`)
Вид операции (приход, расход, ценники, ивентаризация и д.р.)

| № Поля | Тип       | Обязательное | Описание                            | Пример значения |
|-------:|-----------|--------------|-------------------------------------|-----------------|
|      1 | `string`  | Да           | Код операции                        | "01"            |
|      2 | `string`  | Нет          | Код операции для выгрузки документа | ""              |
|      3 | `string`  | Да           | Наименование операции               | "Приход"        |
|      4 | `string`  | Нет          | Код вида аналитики 1                | "ПС"            |
|      5 | `string`  | Нет          | Код вида аналитики 2                |                 |
|      6 | `string`  | Нет          | Код вида аналитики 3                |                 |
|      7 | `integer` | Нет          | Флаги                               | 64            |

|  `preferences` | `VopPreferences` | Нет          | Настройки операции                       |                 |

```kotlin
O;"03";;"Приход";"ПС";;;4
```

### Ключ (`OP`)
Настройка операции

| № Поля | Тип      | Обязательное | Описание     | Пример значения |
|-------:|----------|--------------|--------------|-----------------|

|      2 | `string` | Да           | Имя ключа    | "groupItems"    |

### Ключ (`G`)
Товар

| № Поля | Тип       | Обязательное | Описание                      | Пример значения                        |
|-------:|-----------|--------------|-------------------------------|----------------------------------------|
|      1 | `string`  | Да           | Штрихкод                      | "4810268024917"                        |
|      2 | `string`  | Да           | Наименование                  | "Молоко пастеризованное, 3.1%, 1 литр" |
|      3 | `double`  | Нет          | Цена                          | 3.35                                   |
|      4 | `double`  | Нет          | Остаток                       | 10                                     |
|      5 | `string`  | Нет          | Доп. поле 1                   | "Беларусь"                             |
|      6 | `string`  | Нет          | Доп. поле 2                   | "ОАО Савушкин продукт"                 |
|      7 | `string`  | Нет          | Доп. поле 3                   | ""                                     |
|      8 | `string`  | Нет          | Доп. поле 4                   | ""                                     |
|      9 | `string`  | Нет          | Доп. поле 5                   | ""                                     |
|     10 | `string`  | Нет          | Имя файла изображения         | "4810268024917.jpg"                    |
|     11 | `double`  | Нет          | Процент доверительной приемки | 10.0                                   |
|     12 | `integer` | Нет          | Флаги  (сумма флагов)         | 64                                     |

`Флаги`:
- 1 - Обязательный ввод даты 1
- 2 - Обязательный ввод даты 2
- 4 - Обязательный ввод дополнительного поля 1
- 8 - Обязательный ввод дополнительного поля 2
- 16 - Обязательный ввод дополнительного поля 3
- 32 - Признак товара интернет магазина
- 64 - Признак маркированного товара
- 256 - Признак акционного товара
- 1024 - Подлежит прослеживанию

```kotlin
G;4810268024917;"Молоко пастеризованное, 3.1%, 1 литр";3.35;10;"БеларусьДобрада";"ОАО Савушкин продукт";;;;"4810268024917.jpg";10;64
```

### Объект (`Order`)
Заголовок заказа

| Имя ключа    | Тип                    | Обязательное | Описание                 | Пример значения       |
|--------------|------------------------|--------------|--------------------------|-----------------------|
| `date`       | `string`               | Да           | Дата заказа (ISO 8601)   | "2026-06-15T11:00:00" |
| `number`     | `string`               | Да           | Номер заказа             | "123"                 |
| `idSupplier` | `string`               | Нет          | Код операции `Ana.idAna` | "ПС0001"              |
| `details`    | array of `OrderDetail` | Нет          | Строки заказа            |                       |

### Объект (`OrderDetail`)
Строка заказа

| Имя ключа       | Тип               | Обязательное | Описание                      | Пример значения      |
|-----------------|-------------------|--------------|-------------------------------|----------------------|
| `barcode`       | `string`          | Да           | Штрихкод товара               | "4810268024917"      |
| `quantity`      | `double`          | Нет          | Количество                    | "3"                  |
| `price`         | `double`          | Нет          | Цена                          | "3.35"                   |

```json
{
  "date":  "2026-06-25T11:00:00", 
  "number": "123",
  "idSupplier": "ПС0001", 
  "details": [
    { "barcode": "4810268024917", "quantity": 3, "price": 3.35 },
    { "barcode": "4810268020827", "quantity": 1, "price": 2.53 }
  ]
}
```

### Объект (`Batch`)
Партия

| Имя ключа    | Тип      | Обязательное | Описание                     | Пример значения       |
|--------------|----------|--------------|------------------------------|-----------------------|
| `idBatch`    | `string` | Да           | Код партии                   | "1"                   |
| `number`     | `string` | Нет          | Номер                        | "345"                 |
| `date`       | `string` | Нет          | Дата (ISO 8601)              | "2026-06-26T15:00:00" |
| `barcode`    | `string` | Да           | Штрихкод                     | "4810268024917"       |
| `idSupplier` | `string` | Нет          | Код поставщика (`Ana.idAna`) |                       |
| `price`      | `double` | Нет          | Цена                         | "3.35"                |
| `quantity`   | `double` | Нет          | Количетво                    | "10"                  |


```json
{
  "idBatch": "4810268024917", 
  "name": "Молоко пастеризованное, 3.1%, 1 литр",
  "price": 3.35,
  "quantity": 10,
  "weight": false,
  "image": "4810268024917.jpg",
  "flags": 64
}
```

## Документ ТСД

### Объект (`Document`)
Заголовок документа

| Имя ключа        | Тип                       | Обязательное | Описание                    | Пример значения       |
|------------------|---------------------------|--------------|-----------------------------|-----------------------|
| `date`           | `string`                  | Да           | Дата документа (ISO 8601)   | "2026-06-27T15:00:00" |
| `number`         | `string`                  | Да           | Номер документа             | "12345"               |
| `idVop`          | `string`                  | Да           | Код операции `Vop.idVop`    | "01"                  |
| `idAna1`         | `string`                  | Нет          | Код аналитики 1 `Ana.idAna` | "ПС0001"              |
| `idAna2`         | `string`                  | Нет          | Код аналитики 2 `Ana.idAna` |                       |
| `idAna3`         | `string`                  | Нет          | Код аналитики 3 `Ana.idAna` |                       |
| `comment`        | `string`                  | Нет          | Комментарий                 | "тест"                |
| `numberOrders`   | `string`                  | Нет          | Номер заказа                | "555"                 |
| `details` | array of `DocumentDetail` | Нет          | Строки документа            |                       |

| `price`         | `double`          | Да           | Цена                          | "0"                   |
| `npp`           | `integer`         | Нет          | Порядковый номер              | "0"                   |
| `comment`       | `string`          | Нет          | Комментарий                   |                       |

| `date2`         | `string`          | Нет          | Дата 2 (ISO 8601)             | "2026-06-28"          |
| `field1`        | `string`          | Нет          | Поле 1                        | "test"                |
