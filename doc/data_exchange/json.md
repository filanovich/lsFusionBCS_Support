# Описание JSON обмена

Для обмена данными с терминалом сбора данных (ТСД) предлагается следующий формат на основе JSON-файлов.
Он разделен на два основных потока: загрузка справочной информации в ТСД и выгрузка созданных документов.

## Справочная информация для загрузки в ТСД

| Имя ключа | Тип              | Описание               |
|-----------|------------------|------------------------|
| `van`     | array of `Van`   | Список видов аналитик  |
| `ana`     | array of `Ana`   | Список аналитик        |
| `vop`     | array of `Vop`   | Список видов операций  |
| `item`    | array of `Item`  | Список товаров         |
| `order`   | array of `Order` | Список заказов         |
| `batch`   | array of `Batch` | Список партий          |
| `lot`    | array of `Lot`   | Список марок           |


Пример JSON-файла справочной информации

```json
{
  "van": [
    { "idVan": "ПС", "name": "Поставщик" },
    { "idVan": "ЗН", "name": "Зона" },
    { "idVan": "КМ", "name": "Комиссия" }
  ],
  "ana": [
    { "idAna": "ПС0001", "name": "Поставщик 1" },
    { "idAna": "СК0001", "name": "Склад 1" },
    { "idAna": "ЗН01", "name": "Зона 1" },
    { "idAna": "ЗН02", "name": "Зона 2" },
    { "idAna": "КМ01", "name": "Комиссия 1" },
    { "idAna": "КМ02", "name": "Комиссия 2" }
  ],
  "vop": [
    { "idVop": "01", "name": "Приход", "idVan1": "ПС", "preferences": { "groupItems": true, "isMarking": true } },
    { "idVop": "02", "name": "Инвентаризация", "idVan1": "КМ", "idVan2": "ЗН", "preferences": { "groupItems": true } }
  ],
  "item": [
    {
      "barcode": "4810268024917",
      "name": "Молоко пастеризованное, 3.1%, 1 литр",
      "price": 3.35,
      "quantity": 10,
      "weight": false,
      "image": "4810268024917.jpg",
      "flags": 64
    },
    {
      "barcode": "4810268020827",
      "name": "Кефир \"Савушкин продукт\", 1,5%, (950 г)",
      "price": 2.53,
      "quantity": 5,
      "weight": false,
      "image": "4810268020827.jpg",
      "flags": 64
    }
  ],
  "order": [],
  "batch": [],
  "lot": []
}
```

### Объект (`Van`)
Вид аналитики (поставщик, зона, комиссия и др.) 

| Имя ключа | Тип      | Обязательное | Описание                    | Пример значения |
|-----------|----------|--------------|-----------------------------|-----------------|
| `idVan`   | `string` | Да           | Код вида аналитики          | "ПС"            |
| `name`    | `string` | Да           | Наименование вида аналитики | "Поставцик"     |


```json
{
  "idVan": "ПС",   
  "name": "Поставщик"
}
```

### Объект (`Ana`)
Аналитика

| Имя ключа | Тип      | Обязательное | Описание               | Пример значения                      |
|-----------|----------|--------------|------------------------|--------------------------------------|
| `idAna`   | `string` | Да           | Код аналитики          | "ПС0001" (`Van.idVan` + `Ana.idAna`) |
| `name`    | `string` | Да           | Наименование аналитики | "Поставщик 1"                        |

```json
{
  "idAna": "ПС0001", 
  "name": "Поставщик 1"
}
```

### Объект (`Vop`)
Вид операции (приход, расход, ценники, ивентаризация и д.р.)

| Имя ключа      | Тип              | Обязательное | Описание                                 | Пример значения |
|----------------|------------------|--------------|------------------------------------------|-----------------|
| `idVop`        | `string`         | Да           | Код операции                             | "01"            |
| `name`         | `string`         | Да           | Наименование операции                    | "Приход"        |
| `idVan1`       | `string`         | Нет          | Код вида аналитики 1                     | "ПС"            |
| `idVan2`       | `string`         | Нет          | Код вида аналитики 2                     |                 |
| `idVan3`       | `string`         | Нет          | Код вида аналитики 3                     |                 |
| `idDetailVan1` | `string`         | Нет          | Код вида аналитики для строк документа 1 |                 |
| `idDetailVan2` | `string`         | Нет          | Код вида аналитики для строк документа 2 |                 |
| `preferences`  | `VopPreferences` | Нет          | Настройки операции                       |                 |

```json
{
  "idVop": "01",  
  "name": "Приход",
  "idVan1": "ПС",
  "preferences": { "groupItems": true }
}
```

### Объект (`Item`)
Товар 

| Имя ключа      | Тип              | Обязательное | Описание                | Пример значения                        |
|----------------|------------------|--------------|-------------------------|----------------------------------------|
| `barcode`      | `string`         | Да           | Штрихкод                | "4810268024917"                        |
| `name`         | `string`         | Нет          | Наименование            | "Молоко пастеризованное, 3.1%, 1 литр" |
| `price`        | `double`         | Нет          | Цена                    | "3.35"                                 |
| `quantity`     | `double`         | Нет          | Остаток                 | "10"                                   |
| `weight`       | `boolean`        | Нет          | Признак весового товара | "false"                                |
| `image`        | `string`         | Нет          | Имя файла изображения   | "4810268024917.jpg"                    |
| `flags`        | `integer`        | Нет          | Флаги (сумма флагов)    | "64"                                   |

`flags`:
- 1 - Обязательный ввод даты 1
- 2 - Обязательный ввод даты 2
- 4 - Обязательный ввод дополнительного поля 1
- 8 - Обязательный ввод дополнительного поля 2
- 16 - Обязательный ввод дополнительного поля 3
- 32 - Признак товара интернет магазина
- 64 - Признак маркированного товара
- 256 - Признак акционного товара
- 1024 - Подлежит прослеживанию


```json
{
  "barcode": "4810268024917", 
  "name": "Молоко пастеризованное, 3.1%, 1 литр",
  "price": 3.35,
  "quantity": 10,
  "weight": false,
  "image": "4810268024917.jpg",
  "flags": 64
}
```

### Объект (`Order`)
Заголовок заказа

| Имя ключа    | Тип                    | Обязательное | Описание                 | Пример значения       |
|--------------|------------------------|--------------|--------------------------|-----------------------|
| `date`       | `string`               | Да           | Дата заказа (ISO 8601)   | "2026-06-15T11:00:00" |
| `number`     | `string`               | Да           | Номер заказа             | "123"                 |
| `idSupplier` | `string`               | Нет          | Код операции `Ana.idAna` | "ПС0001"              |
| `details`    | array of `OrderDetail` | Нет          | Строки заказа            |                       |

### Объект (`OrderDetail`)
Строка заказа

| Имя ключа       | Тип               | Обязательное | Описание                      | Пример значения      |
|-----------------|-------------------|--------------|-------------------------------|----------------------|
| `barcode`       | `string`          | Да           | Штрихкод товара               | "4810268024917"      |
| `quantity`      | `double`          | Нет          | Количество                    | "3"                  |
| `price`         | `double`          | Нет          | Цена                          | "3.35"                   |

```json
{
  "date":  "2026-06-25T11:00:00", 
  "number": "123",
  "idSupplier": "ПС0001", 
  "details": [
    { "barcode": "4810268024917", "quantity": 3, "price": 3.35 },

}
```

## Документ ТСД

### Объект (`Document`)
Заголовок документа

| Имя ключа        | Тип                       | Обязательное | Описание                    | Пример значения       |
|------------------|---------------------------|--------------|-----------------------------|-----------------------|
| `date`           | `string`                  | Да           | Дата документа (ISO 8601)   | "2026-06-27T15:00:00" |
| `number`         | `string`                  | Да           | Номер документа             | "12345"               |
| `idVop`          | `string`                  | Да           | Код операции `Vop.idVop`    | "01"                  |
| `idAna1`         | `string`                  | Нет          | Код аналитики 1 `Ana.idAna` | "ПС0001"              |
| `idAna2`         | `string`                  | Нет          | Код аналитики 2 `Ana.idAna` |                       |
| `idAna3`         | `string`                  | Нет          | Код аналитики 3 `Ana.idAna` |                       |
| `comment`        | `string`                  | Нет          | Комментарий                 | "тест"                |
| `numberOrders`   | `string`                  | Нет          | Номер заказа                | "555"                 |
| `details` | array of `DocumentDetail` | Нет          | Строки документа            |                       |

### Объект (`DocumentDetail`)

| Имя ключа       | Тип               | Обязательное | Описание                      | Пример значения       |

| `quantity`      | `double`          | Да           | Количество                    | "3"                   |
| `extraQuantity` | `double`          | Да           | Доп. количество               | "0"                   |
| `price`         | `double`          | Да           | Цена                          | "0"                   |
| `npp`           | `integer`         | Нет          | Порядковый номер              | "0"                   |
| `comment`       | `string`          | Нет          | Комментарий                   |                       |
| `date1`         | `string`          | Нет          | Дата 1 (ISO 8601)             | "2026-06-27"          |
